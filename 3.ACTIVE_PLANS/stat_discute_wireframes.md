## 1. Structure globale d'une page d'analyse

### 1.1 Layout général

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                                    HEADER                                        │
│  [Logo]                    [Sommaire cliquable]                    [Assistant]   │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                  │
│                                                                                  │
│                                                                                  │
│                            CONTENU SÉQUENCE                                      │
│                            (100vw × ~85vh)                                       │
│                                                                                  │
│                                                                                  │
│                                                                                  │
│                                                                                  │
├─────────────────────────────────────────────────────────────────────────────────┤
│                              ○  ○  ●  ○  ○  ○                                   │
│                              DOTS PAGINATION                                     │
└─────────────────────────────────────────────────────────────────────────────────┘
```

### 1.2 Header détaillé

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                                                                                  │
│   ┌────────┐                                                      ┌───────────┐ │
│   │  LOGO  │     LAL @ BOS  ·  Totaux  ·  O/U 226.5              │ Assistant │ │
│   └────────┘                                                      └───────────┘ │
│                                                                                  │
│              [Contexte] [Scoring LAL] [Scoring BOS] [Combiné] [Synthèse]        │
│                  ▲                                                               │
│              actif                                                               │
└─────────────────────────────────────────────────────────────────────────────────┘
```

### 1.3 Dots de pagination

```
                              ←  ○  ○  ●  ○  ○  ○  →
                                       ▲
                                   actif

Comportement :
- Clic sur dot = navigation directe
- Flèches optionnelles aux extrémités
- Dot actif = rempli ou plus grand
```
